MIH Lab 2016 Releases

Demo Machine: Make your Odigo Omnichannel Widget on Any Website

More than 200 french and international projects / demos created since release in early 2016

Available in 6 languages
Already integrated with Salesforce's digital channels (chat, email)
and with Zendesk’s chat and FAQ platform

Customer Info on Odigo Using Dimelo API

Up-to-date information and interactions history of a customer in Odigo

Now supporting chat and sms interactions, and available in English
At first an experimental interface for a Casden oportunity, using the customer data inside Dimelo;
it has now been used on a great number of agents for french and international opportunities

Concierge Chatbot Front-End Interface

Relooking and Front-End developments for the New Concierge Chatbot,
currently being tested by Pôle-Emploi

Pôle Emploi Mockups & Demo Interfaces

in collaboration with Christelle Anglade & Thomas Saint Hilaire

Creation of Demo Interfaces integrated with the New Odigo Console
